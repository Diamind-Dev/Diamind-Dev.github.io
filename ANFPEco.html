What Is ANFP Eco?

ANFP Eco is a streamlined version of ANFP, our Unreal Engine plugin designed specifically for Android devices. It offers the simplest way to select files from storage within a game, returning the file path as a string variable. This absolute file path output makes it highly compatible with importer plugins and other development tools.

We offer the plugin in two versions:

  • ANFP(Full Version) : Includes all advanced features and functionalities.
  • ANFP Eco : A simpler, budget-friendly version that retains the core functionality at half the price.

Features

ANFP Eco offers all the basic features, including access to all file types within local storage, except for Google Drive and other external apps like the gallery, music player, and file manager that provide a better UI/UX. In this version, all options are predefined and cannot be modified — but these core features might be all you need.
Here’s a concise list of them:

  • Multi file selection, select all
  • Search functionality
  • Sort files by name, type, size, and time
  • Thumbnail for images

Most features are supported on almost all Android devices, but some Android editions and devices may support more or fewer features or have different UI designs (since the file picker is a built-in Android app).



Set up ANFP Eco

Setting Up ANFP Eco is quick and easy, requiring only 7 blueprint nodes to get everything running smoothly. Our step-by-step guide ensures that you’ll have the file picker integrated in no time.
