Android Native File Picker

ANFP is an unreal engine plugin specified for android devices, provides the greatest design and the easiest way to choose a file from storage through game and returns the path of selection files as a string variable for further usage like importing files into your game (the ANFP doesnâ€™t import files on its own!).

ANFP covers a wide range of file types. it almost supports all categories of files and even lets developers modify compatible files through a very simple blueprint approach .For more details, see supported files.

ANFP is compatible with android 8 and later. it only supports android 64-bit devices and now is available for UE 5.2_5.4 on marketplace.

How To Install

ANFP is now available on the Epic Games Marketplace and will be automatically installed in your project once the download is complete.
